Это Русская игра про любовь 2-ух парней. Все имена изменены в целях сохранить конфедециальность. Тут есть чать правды а часть выдумки.

Управление: 

1. кнопки - выбор действия
2. Кнопка справа в углу (Значек Домика) - Главное меню

Чтобы начать играть сделайте эти пункты:
1. скачайте папку Nove
2. откройте папку dist
3. нажмите 2 раза ЛКМ на EXE файл
